## dotenv
1. Create a new file name `.env`
2. Write a message according to the example
```bash
DATABASE_URI = 'mongodb api'

NODE_ENV = "start"

ACCESS_TOKEN_SECRET = "supersecretkey_example"
REFRESH_TOKEN_SECRET = "refreshsecretkey_example"
PORT = 8080
```
